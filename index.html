Кто такие «Саппорты»?

Саппорт — герой, который помогает раскачаться своему керри. Саппорты слабо зависимы от вещей. Задача — сделать фарм союзников приятным, а врагов — невыносимым.

🎥 YouTube: Основы саппорта

⚡

Что должен делать Support

1. Бебиситинг

Саппорт должен спасать керри даже ценой своей жизни.

2. Харасс

Мешать врагу фармить, наносить урон. Дуэль саппортов.

3. Пуллинг

Удерживать крипов ближе к башне через отвод к нейтралам.

4. Денай

Добивать своих крипов ниже 50% ХП, чтобы враг не получал опыт.

🗺️

Расстановка героев

2–1–2

Мидер соло, саппорт с керри, второй с оффлайнером.

3–1–1

Тройка на тяжёлой линии давит врага.

1–1–3

Тройка на безопасной для пуша.

1–1–2 + лес

С Chen, Enchantress.

🛒

Стартовый закуп

  • 2 Observer Wards
  • 1 Courier
  • Tango, Salve, Clarity, веточки
⚔️

Тактика и контроль

В замесах — станы, хил, дизэйбл. Не умирайте первым.

Кто такие «CARRY»?

Carry — герои, которые приносят победу. Зависят от предметов. В начале слабы, в конце — могущественны.

🏆

Semi-Carry и Hard Carry

Hard-Carry — сильны в поздней игре, способны убивать нескольких врагов с хорошими артефактами.

📈

Поздняя игра

  • Нужен фарм для предметов
  • Пассивные скиллы требуют уровней
  • Приоритет — ловкость: скорость атаки и броня
🎯

Как играть за Carry

80+ крипов к 10 минуте — идеал. 4-5/мин — хорошо. Тренируйте ластхит.

📊

Характеристики

Сила — танки, ХП. Ловкость — скорость атаки. Интеллект — магия.

🗺️

Линии

Safe lane — безопасно для керри. Hard lane — сложнее, больше опыта.

Максимальная оптимизация Dota 2

Полный гайд по оптимизации: параметры запуска, настройки Windows, утилиты и автоэкзек.

🚀

Параметры запуска

Зайдите в Свойства Dota 2 в Steam → Параметры запуска и вставьте нужные команды:

  • -novidПропустить заставку Valve
  • +fps_max 120Ограничить FPS (укажите своё число)
  • -prewarmПредзагрузка данных (HDD)
  • +map_enable_portrait_worlds 0Убрать эффекты в меню
  • -noaafontsОтключить сглаживание шрифтов
  • -novrОтключить VR
  • -nojoyОтключить джойстик
  • -highВысокий приоритет процесса
  • -nod3d9exОтключить расширения Aero
  • -noipxСнизить нагрузку на ОЗУ
  • -threads 2/4Количество ядер CPU
  • -dx9 / -dx11 / -vulkan / -glВыбор API
  • -32bit32-битный клиент (меньше RAM)
  • -autoconfigСбросить настройки до стандарта
  • -fullscreen / -windowed / -noborderРежим экрана
  • -map dotaЗагружать карту при старте
🔧

Выбор API: Vulkan vs DirectX

Vulkan — для современных видеокарт (особенно AMD), многоядерных CPU. Лучше стабильность FPS.

DirectX 11 — для старого оборудования и некоторых NVIDIA. Стабильнее на Windows.

Рекомендация: протестируйте оба API в одинаковых условиях и выберите лучший.

Чтобы включить Vulkan: Свойства → Доп. контент → установить Vulkan, затем выбрать в настройках графики.

Оптимизация Windows

1. Чистка автозагрузки

Уберите Razer Synapse, Logitech G Hub и прочие ненужные программы из автозагрузки.

2. Visual C++ Redistributable

Скачайте и установите с techpowerup.com — запустите install_all.bat от админа.

3. Игровой режим

Параметры → Игры → Игровой режим → Вкл.

4. План электропитания

powercfg -duplicatescheme e9a42b02-d5df-448d-aa00-03f14749eb61 (админская CMD) → включить «Максимальная производительность».

5. Отключение Game Bar

Win 10: Параметры → Игры → Game Bar. Win 11: Get-AppxPackage Microsoft.XboxGamingOverlay | Remove-AppxPackage

Также отключите аппаратное ускорение и оверлей в Discord.

6. Драйвера

Обновите драйвера видеокарты через GeForce Experience / AMD Radeon Software.

Оптимизация Dota 2

Настройки графики

Выставьте минимальные настройки. Можно включить FidelityFX для апскейлинга.

Minify — программа для оптимизации

Безопасно, не банят. Открытый код на GitHub.

  1. Установите Dota Workshop Tools (Свойства → Доп. контент)
  2. В параметры запуска добавьте -language russian
  3. Скачайте Minify с GitHub
  4. Запустите Minify.exe от админа, выберите галочки, нажмите Patch
  5. Переместите файлы из dota_minify в dota_russian

Важно: После обновления Dota 2 мод может сломаться — скачайте новую версию Minify и установите заново.

Оптимизация Steam

Отключите в настройках Steam всё лишнее: загрузки во время игры, анимации, оверлей, автоматические обновления.

Максимальная оптимизация Steam (.bat)

Создайте файл steam_opt.bat с содержимым:

taskkill /f /im steam.exe >nul 2>&1
rd /s /q "%localappdata%\Steam\htmlcache"
rd /s /q "%localappdata%\Steam\shaders"
rd /s /q "%localappdata%\Steam\cache"

start "" "C:\Program Files (x86)\Steam\steam.exe" ^
  -noverifyfiles ^
  -quicklogin ^
  -no-browser ^
  -nofriendsui ^
  -skipstreamingdrivers ^
  -cef-single-process ^
  -cef-disable-gpu ^
  -cef-disable-gpu-compositing ^
  -cef-disable-breakpad ^
  -cef-force-occlusion ^
  -no-dwrite ^
  -vrdisable ^
  -single_core ^
  -nocrashdialog

Измените путь к Steam.exe под вашу систему. Вместо дефолтных 10 процессов — будет всего 1.
